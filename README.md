# 🍽️ Recipe App

A simple and modern Recipe Search Application built using **HTML, CSS, and JavaScript**.  
The app fetches real recipes from an external API and displays them in a clean UI.

---

## 🚀 Features

- 🔍 Search for recipes by name
- 📡 Fetch real data from API (TheMealDB)
- 🧾 View recipe details and instructions
- 🍲 Display ingredients dynamically
- 🎨 Responsive and clean UI
- ⚡ Fast and lightweight (no frameworks)

---

## 🛠️ Technologies Used

- HTML5
- CSS3 
- JavaScript (ES6)
- Fetch API
- TheMealDB API
